Enum combine::easy::Info

source · 
pub enum Info<T, R> {
    Token(T),
    Range(R),
    Owned(String),
    Static(&'static str),
}
Expand description

Enum holding error information. Variants are defined for Stream::Token and Stream::Range as well as string variants holding easy descriptions.

As there is implementations of From for String and &'static str the constructor need not be used directly as calling msg.into() should turn a message into the correct Info variant.
