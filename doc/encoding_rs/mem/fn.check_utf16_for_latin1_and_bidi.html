pub fn check_utf16_for_latin1_and_bidi(buffer: &[u16]) -> Latin1Bidi
Expand description

Checks whether a potentially invalid UTF-16 buffer contains code points that trigger right-to-left processing or is all-Latin1.

Possibly more efficient than performing the checks separately.

Returns Latin1Bidi::Latin1 if is_utf16_latin1() would return true. Otherwise, returns Latin1Bidi::Bidi if is_utf16_bidi() would return true. Otherwise, returns Latin1Bidi::LeftToRight.
