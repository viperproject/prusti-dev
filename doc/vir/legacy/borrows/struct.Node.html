Node of the reborrowing DAG.

Fields

guard: Expr

The basic block at which the borrow occured was executed only iff the guard is true.

borrow: Borrowreborrowing_nodes: Vec<Borrow>reborrowed_nodes: Vec<Borrow>stmts: Vec<Stmt>borrowed_places: Vec<Expr>

Borrows that are borrowing the same place.
