A local variable.

An enum variant.

Field(Field)

A field access. (Field accesses can never fail.)

Deref(Deref)

A reference or pointer dereference. (Sometimes can fail.)

Downcast(Downcast)

Inform the fold-unfold algorithm that at this program point a enum type can be downcasted to one of its variants. This statement is a no-op for Viper. Arguments:

  • expression in which the downcast is visible
  • place to the enumeration that is downcasted
  • field that encodes the variant
