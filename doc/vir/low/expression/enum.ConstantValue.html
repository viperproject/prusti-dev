Enum vir::low::expression::ConstantValue

source · 
pub enum ConstantValue {
    Bool(bool),
    Int(i64),
    BigInt(String),
}
Expand description

Note: we do not have explicit constant values for bit vectors and floating points because they are stored as Int/BigInt and translated into Viper based on the Constant.ty field.
