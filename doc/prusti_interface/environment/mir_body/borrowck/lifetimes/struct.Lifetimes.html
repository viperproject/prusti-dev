pub fn construct_replacement_map(
    &self,
    location: Location,
    base_lifetimes: Vec<LifetimeConst>
) -> BTreeMap<LifetimeConst, LifetimeConst>

When the Rust compiler generates Polonius facts, it creates many temporary lifetimes. Sometimes (for example, when encoding the constructing assignment of the aggregate with a reference-typed field) we need to unify the lifetimes. This method constructs a map from lifetimes known to Polonius to base_lifetimes indicating with which base_lifetimes the lifetimes could be replaced.
