fn split_assignment(
    self,
    tcx: TyCtxt<'tcx>,
    mir: &Body<'tcx>
) -> Vec<Statement<'tcx>> ⓘ

Transforms an assignment into its atomic parts. For a normal assignment like

ⓘ
    _3 = move _2;

this is simply returning the assignment itself. But for an aggregate assignment like

ⓘ
    _3 = (move _1, move _2);

there is one atomic assignment for every tuple field. In this example, the atomic assignments would be

ⓘ
    _3.0 = move _1;
    _3.1 = move _2;

Statements that are no assignments are returned untouched.
