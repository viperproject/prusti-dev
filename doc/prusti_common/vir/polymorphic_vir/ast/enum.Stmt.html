TransferPerm(TransferPerm)

Tuple Fields

0: TransferPerm

Move permissions from a place to another. This is used to restore permissions in the fold/unfold state when a loan expires.

The last argument indicates if the transfer is unchecked. Unchecked transfer is used for encoding shared borrows which can be dangling and, therefore, we cannot use the safety checks.
