pub fn build<T>(
    self,
    password: &str,
    friendly_name: &str,
    pkey: &PKeyRef<T>,
    cert: &X509Ref
) -> Result<Pkcs12, ErrorStack>where
    T: HasPrivate,

Builds the PKCS #12 object

Arguments
  • password - the password used to encrypt the key and certificate
  • friendly_name - user defined name for the certificate
  • pkey - key to store
  • cert - certificate to store

This corresponds to PKCS12_create.
