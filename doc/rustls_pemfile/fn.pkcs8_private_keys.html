pub fn pkcs8_private_keys(rd: &mut dyn BufRead) -> Result<Vec<Vec<u8>>, Error>
Expand description

Extract all PKCS8-encoded private keys from rd, and return a vec of byte vecs containing the der-format contents.

This function does not fail if there are no keys in the file – it returns an empty vector.
